# datasets/

Versioned ML training datasets for Phase 2+ model development.

Each version subdirectory (v1.0, v1.1, etc.) contains:
- `train_*.parquet` — training set
- `val_*.parquet` — validation set
- `test_*.parquet` — test set
- `features_schema.json` — feature definitions
- `labels_schema.json` — target definitions

Generated by `scripts/data_store/curate_dataset.py` post-live-trading.

See ../README.md for full documentation.
