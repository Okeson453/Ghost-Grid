# performance/

Trade performance summaries and metrics aggregation.

Subdirectories:
- `daily/` — daily P&L and trade counts (one JSON per day)
- `monthly/` — monthly aggregated reports
- `regime_analysis/` — performance broken down by regime, session, score band

Generated by `scripts/data_store/compute_performance_summary.py` daily.

Used for:
- Equity curve monitoring
- Win rate tracking
- Regime-specific performance analysis
- Model hyperparameter optimization

See ../README.md for full documentation.
